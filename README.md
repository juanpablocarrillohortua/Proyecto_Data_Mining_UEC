# CRISP-DM - Spotify

## Descripción del proyecto
Este proyecto es una demostración de las fases propias de CRISP-DM, con especial enfasis en las 3 fases iniciales, el proyecto se basará en una base de datos extraida de kaggle, la cual recopila datos de canciones obtenidos desde la API de spotify.

### Créditos y Origen de los Datos

Este proyecto utiliza un conjunto de datos obtenido originalmente de **Kaggle**, cuyos contenidos provienen de la **Spotify Web API**.

* **Autor del Dataset:** [Joakim Arvidsson](https://www.kaggle.com/joebeachcapital)
* **Enlace de Kaggle:** [30000 Spotify Songs](https://www.kaggle.com/datasets/joebeachcapital/30000-spotify-songs)
* **Fuente Primaria:** [Spotify Developer Platform](https://developer.spotify.com/)
